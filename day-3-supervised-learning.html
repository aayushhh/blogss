游누Day 3 Supervised Learning

Aayush March 6, 2021 31daysofML

There are different types of learning models. These 4 are the most common that you usually come across.

  • Supervised
  • Unsupervised
  • Semi-supervised
  • Reinforcement



  • Supervised learning - Applications in which the training data is examples of the input vectors along with their corresponding target vectors

2 main types:

Classification: Predicting a class label

Regression: Predicting a numerical label

  • Unsupervised learning is using a model to describe or extract relationships in data. Operates on only the input data without target variables

2 main types:

Clustering: Finding groups in data

Density Estimation: Summarizing distribution of data

  • Semi-supervised - training data contains very few labeled examples & large number of unlabeled examples.

The goal is to make effective use of all of the available data.

A mixture of supervised & unsupervised methods are used.

  • Reinforcement learning - an agent operates in an environment & learns to operate using feedback.

There's no fixed training dataset, just a set of goals that an agent is set to achieve & there is a feedback loop between the learning system and its experiences.

2 types of supervised learning

Regression model predicts continuous values

Value of house in LA?

What's the probability of stock value to rise?

Classification model predicts discrete values

Is this image cat or dog?

Is this email spam?

Regression is form of predictive analysis. It is used for:

Forecasting - estimate growth in sales

Time-series - closing value of stock exchange

Finding cause effect relationships - effect of price change on sales

most common regression types are:

Linear Regression - Establishes a relationship between dependent variable (Y) and one or more independent variables (X) using a best fit straight line

Logistic Regression - Used to find probability of an event

Logistic regression estimates probability as output which we can use in 2 ways:

"As is" - probability that a dog will bark at night eg: 0.06

Converted to a binary category - for a classification problem. "Email is a spam or not" == 0 or 1

Classification models predicts the class to which input data elements belongs. It is best used when the output has finite and discrete values.

Email is spam or not

Student will pass an exam or now

Classification Algorithms

  • Logistic Regression
  • Support Vector Machines
  • Nonlinear models (multi-class)
  • K-nearest Neighbors (KNN)
  • Kernel Support Vector Machines (SVM)
